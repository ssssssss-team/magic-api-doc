# 聚合函数 0.5.5+

# count

  • 入参：target:Object
  • 返回值：int
  • 函数说明：计算集合大小
var list = [1,2,3,4,5]
return count(list);  // 5

# group_concat

  • 入参：target:Object
  • 入参：separator:String 分隔符，可省略
  • 返回值：Object
  • 函数说明：将集合拼接起来
var list = [1,2,3,4,5]
return group_concat(list);  // "1,2,3,4,5"
// return group_concat(list,'|');  // "1|2|3|4|5"
