# 扩展说明

# 目的

目的只有一个：为已有类型添加方法，如给Object类型添加asInt方法

# 使用

所有类型扩展使用都很简单，无需import直接使用 如Object类的扩展。

var obj = '123';
return obj.asInt(); //调用Object的扩展方法asInt
